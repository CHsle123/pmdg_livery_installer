# PMDG Livery Installer
# Features
- Install liveries in .zip (OC3 format) and .ptp formats to the Community folder.
- Supports custom installation directories with symbolic links created in the Community folder (convenient for MSFS Addons Linker users).
- Supports direct conversion of .ptp liveries to the MSFS 2024 format (Currently supports the 738, 77W, and 77F aircraft models).
- Supports uninstallation of PMDG liveries.
- Automate the update of the layout.json file.
- Unknown issues may exist; feedback is welcome.
# Compability
MSFS 2024
